## 📌 Descrição
Este projeto em C++ permite gerir notas de alunos, calcular médias e gerar uma pauta final automaticamente.

## ⚙️ Funcionalidades
- Leitura de nomes a partir de ficheiro
- Inserção de notas
- Cálculo de média
- Indicação de aprovado/reprovado
- Geração do ficheiro pauta_final.txt

## ▶️ Como executar
1. Compilar:
g++ main.cpp -o programa

2. Executar:
./programa

## 📊 Critério
- Média ≥ 10 → Aprovado
- Média < 10 → Reprovado
